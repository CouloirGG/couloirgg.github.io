// Features

Built for Raiders

Stop alt-tabbing to wikis and spreadsheets. ARC View puts everything on screen while you play.

📡

Overlay HUD

Always-on-top event timer with active raid countdowns, next event preview, and imminent alerts. See what's happening without leaving the game.

📦

Stash Organizer

Bulk-analyze every item in the database with keep/sell/recycle verdicts. Filter by verdict, search by name, and see crafting uses and recycle yields at a glance.

⚙️

Build Advisor

Pick your playstyle and get ranked gear and skill recommendations. DPS, Survival, Farming, or Balanced — the advisor scores items against your goals.

📊

Raid Analytics

Log every raid and track your extraction rate, average loot value, and loadout performance over time. Pre-raid risk assessment helps you decide what to bring.

🗺️

Map Intel Overlay

OCR-powered map detection that auto-shows threats, quests, events, and loot when you enter a new area. Pin it or let it auto-dismiss.

🔍

Item Database

Full item browser with stat bars, rarity filters, equipment slot grid, and side-by-side comparison. Every weapon, armor, and material in the game.

◆ ◆ ◆
// How It Works

Three Steps

No accounts, no API keys, no configuration. Download ARC View and go.

Download & Launch

Grab the latest release from GitHub. Extract the portable exe and run it. No installer needed.

Play Arc Raiders

ARC View's overlay sits on top of your game. Event timers, quest progress, and map intel update automatically.

Use the Tools

Browse items, run the stash organizer, check build recommendations, or log your raids. All in one window alongside your game.
